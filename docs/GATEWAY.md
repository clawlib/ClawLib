## English

The **Claw Gateway** is what makes your agent accessible across different devices and networks. It acts as a standardized communication layer.

### How it Works

The Gateway runs as a local daemon. It maintains persistent connections to messaging providers and translates their proprietary message formats into the **Claw Unified Message Format**.

### Features

- **Protocol Translation**: Converts platform-specific objects (Telegram, WhatsApp) into a common interface.
- **Media Transcoding**: Resizes images or converts formats to ensure Kernel compatibility.
- **Monitoring**: Provides health checks and system metrics.
